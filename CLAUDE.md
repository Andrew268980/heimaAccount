# 黑马记账 (Heima Accounting)

## 项目信息

- **项目名称**：黑马记账
- **项目类型**：跨平台桌面应用
- **目标平台**：Windows 10+ / macOS 11+
- **技术栈**：Tauri 2.x + React 18 + TypeScript + Tailwind CSS
- **UI 组件库**：shadcn/ui ✅
- **数据存储**：SQLite（通过 Tauri 插件 tauri-plugin-sql）
- **包管理**：pnpm
- **开发语言**：TypeScript (前端) + Rust (Tauri 后端)

## 产品功能

### 核心功能

1. **添加支出记录**：金额（人民币元）、一级分类、二级分类、日期、备注（可选）
2. **账单列表**：按时间倒序展示所有记录，支持分页/无限滚动
3. **编辑/删除**：修改或删除已有记录，删除前确认
4. **月度统计**：按月查看总支出金额、各分类占比（饼图/柱状图）
5. **分类筛选**：按一级或二级分类查看支出明细
6. **数据本地存储**：所有数据存储在本地 SQLite 数据库，无需联网
7. **数据导出**（后续迭代）：导出为 CSV 文件

## UI 设计风格

- **整体风格**：简洁现代，类似手机记账 App 的桌面版
- **布局**：单窗口应用，左侧导航 + 右侧内容区
- **色彩**：以绿色/蓝色为主色调（财务感），搭配中性灰白
- **组件风格**：卡片式布局、圆角设计、柔和阴影
- **字体**：系统默认中文字体（微软雅黑 / PingFang SC）

## 开发协作规则（重要！）

> ⚠️ **用户不是程序员，不懂任何编程技术。在整个项目开发过程中，遇到任何技术决策时，Claude 必须：**

1. **列出至少 2-3 个可选方案**
2. **用通俗语言解释每个方案是什么、有什么优缺点**
3. **给出明确推荐（标注"推荐"）**
4. **等待用户做出选择后，再继续开发**
